# MyPlanNote

## 概要
自分のやること、予定、目標などをメモし、完了したものを可視化するTodoアプリ

## できること
- ユーザー認証(breeze)
- タスクのメモ
- タスクのふりわけ(完了・未完了)
- カテゴリ絞り込み機能
